
Мне уже не больно. <br/>
Вроде бы. <br/>
Я не строю планов на завтра, <br/>
В сумке любимый автор <br/>
И три бутерброда. <br/>
Что-то продано, <br/>
Что-то куплено <br/>
И раздарено. <br/>
Стихи бестолковые. <br/>
Стихи матерные. <br/>
Я бездарен и <br/>
Не прошу понимать меня. <br/>
Не там и не с теми <br/>
Раскрываю душу. <br/>
На улице минус двадцать. <br/>
Я так люблю просыпаться <br/>
В своей постели <br/>
И засыпать <br/>
На своей подушке. <br/>
Мне кажется это <br/>
Самое настоящее счастье. <br/>
Солнце блестит, <br/>
Как новенький пенни, <br/>
Разгоняя хмурые тучи. <br/>
Бог одолжит на завтра <br/>
Немного сил <br/>
И терпения. <br/>
И я справлюсь. <br/>
Все получится <br/>
Ведь мне уже не больно.
