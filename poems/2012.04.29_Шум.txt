
В голове шум. <br/>
Помню закат в Харькове. <br/>
Улыбки в Одессе. <br/>
Помню салаты в пзх и плов. <br/>
Кто-то просил теплее одеться, <br/>
Но я делал вид, что мне тепло <br/>
И я никуда не спешу. <br/>
Слова резали лучше скальпеля. <br/>
Помню погоду ясную. <br/>
Подарки из Красноярска. <br/>
Открытки. <br/>
И разговоры по скайпу. <br/>
В небо, высоко очень, <br/>
Подбрасывал батут. <br/>
Кто-то кормил и вписывал, <br/>
Тратил силы. <br/>
Я не помню имен и отчеств, <br/>
Но я помню ту доброту. <br/>
За нее спасибо. <br/>
Вечерний Киев. Песни в темпе адажио. <br/>
Бабушкины обеды и кислый щавель. <br/>
Книги на распродажах. <br/>
И те червоточины. <br/>
Говори, что хочешь, <br/>
Но я обещал. И держал обещания. <br/>
Не сложилось у меня. <br/>
Ни с людьми. Ни с модой. <br/>
О главном никак. О главном коротко. <br/>
Тяжелые тучи спускаются по крышам заводов. <br/>
На мой уставший город. <br/>
Я шучу и улыбаюсь чинно. <br/>
Как в детстве, у дома цветут акации. <br/>
Мне давно не пятнадцать, <br/>
Но я все так же вру маме. <br/>
Правда, уже по другим причинам. <br/>
Помню даблминт на двоих. И с лебедями озеро. <br/>
Помню тех, кому я был правда нужен, <br/>
Кто подставлял плечо. <br/>
И тех, кто меня просто использовал. <br/>
Всем спасибо. <br/>
Приходите еще. <br/>
Помню смешным себя. Помню серьезным. <br/>
В час ночи про собак передачи по ящику. <br/>
Помню на вокзале слезы. <br/>
Вот они были настоящими. <br/>
Глубина глаз и теплота ладоней. <br/>
Всегда приходит время прощаться. <br/>
Смех без причины - признак счастья. <br/>
Ты просто запомни. <br/>
Как я запомнил. <br/>
Вокзалы. Люди. Киоски. <br/>
Редкие звонки от мамы. <br/>
Я помню так много. <br/>
И так чертовски <br/>
Мало. <br/>
<br/>
В голове. День ото дня. <br/>
Я кручу на репите <br/>
Шум.
